Краткое описание проекта

Название: AI Agent Studio

Описание:
Кастомный веб-интерфейс для управления AI-агентами на основе форка OpenHands. Включает мультиагентную систему (Generator, Critic, Optimizer, Tester, Finalizer), гибкую настройку моделей для каждого агента, и графический установщик для Linux и Windows.

Ключевые особенности:

- Чат, терминал, редактор кода, файловый менеджер, встроенный браузер.
- Интеграция с GitHub (авторизация, коммиты, PR).
- Панель управления агентами (добавление/удаление, порядок, настройка моделей).
- Отображение «мышления» агентов в реальном времени.
- Установщик с выбором локальной/облачной модели, настройкой прокси.

Технологии:

- Frontend: React, TypeScript, Monaco Editor, xterm.js
- Backend: FastAPI, AutoGen, Python
- Установщик: Electron
- Контейнеризация: Docker

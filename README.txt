Краткое описание проекта

Название: AI Agent Studio

Описание:
Кастомный веб-интерфейс для управления AI-агентами на основе форка OpenHands. Включает мультиагентную систему (Generator, Critic, Optimizer, Tester, Finalizer), гибкую настройку моделей для каждого агента, и terminal-first установщик для Linux/Windows. Electron может использоваться только как дополнительный desktop launcher.

Ключевые особенности:

- Чат, терминал, редактор кода, файловый менеджер, встроенный браузер.
- Интеграция с GitHub (авторизация, коммиты, PR).
- Панель управления агентами (добавление/удаление, порядок, настройка моделей).
- Отображение «мышления» агентов в реальном времени.
- Терминальный установщик с выбором локальной/облачной модели, настройкой прокси.

Технологии:

- Frontend: React, TypeScript, Monaco Editor, xterm.js
- Backend: FastAPI, AutoGen, Python
- Установщик: Bash bootstrap + Node.js CLI; Electron только опционально
- Контейнеризация: Docker
